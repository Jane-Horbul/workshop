### Тема 1: Масиви (Arrays) — вже є

#### Рівень 1 — Початковий (поточний)
- Створення масиву `[]`
- Доступ за індексом `arr[0]`
- `.length`
- `.push()`, `.pop()`
- `.includes()`

#### Рівень 2 — Середній
- `.splice()` — видалення/вставка елементів за індексом
- `.slice()` — копіювання частини масиву
- `.indexOf()`, `.findIndex()` — пошук елемента
- `.sort()` — сортування (порівняння рядків vs чисел)
- `.reverse()` — перевертання масиву
- Різниця між `.splice()` і `.slice()`
- Spread-оператор `[...arr]` для копіювання

#### Рівень 3 — Просунутий
- `.reduce()` — акумулювання значень
- `.flat()`, `.flatMap()` — робота з вкладеними масивами
- Деструктуризація масивів `const [a, b] = arr`
- `.every()`, `.some()` — перевірка умов
- Ланцюжки методів `.filter().map().reduce()`
- `Array.from()`, `Array.isArray()`
- Іммутабельність — чому важливо не змінювати оригінал

## Фаза 2: Збереження результатів (localStorage)

### Що зберігаємо

```javascript
// Структура даних в localStorage
{
  "quizResults": [
    {
      "id": "uuid-1234",
      "topic": "arrays",
      "level": 1,
      "correct": 8,
      "wrong": 2,
      "total": 10,
      "percentage": 80,
      "date": "2026-04-01T14:30:00.000Z",
      "timeSpent": 125       // секунди на весь квіз
    }
  ],
  "userProgress": {
    "arrays":    { "bestLevel": 2, "bestScore": [90, 70, 0] },
    "variables": { "bestLevel": 1, "bestScore": [80, 0, 0] },
    "functions": { "bestLevel": 0, "bestScore": [0, 0, 0] },
    "objects":   { "bestLevel": 0, "bestScore": [0, 0, 0] },
    "loops":     { "bestLevel": 0, "bestScore": [0, 0, 0] }
  },
  "settings": {
    "playerName": "Учень",
    "theme": "dark"
  }
}
```

### Функціонал

1. **Автоматичне збереження** — після кожного завершеного квізу
2. **Історія спроб** — список усіх пройдених квізів з датою та результатом
3. **Найкращий результат** — для кожної теми/рівня зберігається рекорд
4. **Розблокування рівнів** — рівень 2 відкривається при >= 70% на рівні 1
5. **Статистика** — загальна кількість пройдених квізів, середній бал
6. **Скидання прогресу** — кнопка для очищення всіх даних
7. **Ім'я гравця** — вводиться один раз, зберігається

### API для роботи з localStorage

```javascript
// Модуль storage.js
const Storage = {
  save(key, data)     // Зберегти дані
  load(key)           // Завантажити дані
  addResult(result)   // Додати результат квізу
  getBestScore(topic, level)  // Найкращий результат
  isLevelUnlocked(topic, level)  // Чи відкритий рівень
  getHistory()        // Вся історія
  clearAll()          // Очистити все
};
```

## Фаза 3: Динамічний дизайн та анімації

### 3.1 Анімовані частинки на фоні (Particles)

- Canvas-елемент на весь екран за контентом
- Плаваючі точки/зірки з повільним рухом
- Лінії між близькими частинками (ефект "сітки")
- Частинки реагують на рух миші — відштовхуються або притягуються
- Колір частинок змінюється залежно від поточної теми квізу

### 3.2 Анімації переходів між питаннями

- **Slide-in** — нове питання "в'їжджає" зліва
- **Fade** — плавне зникнення/появлення
- Прогрес-бар анімовано заповнюється
- Номер питання плавно змінюється з ефектом "flip"

### 3.3 Реакція на відповідь

#### Правильна відповідь:
- Кнопка пульсує зеленим кольором
- Вибух конфетті (canvas-анімація) — 30-50 частинок різного кольору
- Текст "Правильно!" з'являється з bounce-ефектом
- Коротка вібрація на мобільних (Vibration API)
- Зірочка або галочка "влітає" в лічильник правильних відповідей

#### Неправильна відповідь:
- Кнопка трясеться (shake animation)
- Легке червоне мерехтіння фону
- Правильна відповідь підсвічується з пульсацією
- Хрестик з'являється з fade-in

### 3.4 Екран результатів

- Числа правильних/неправильних — анімований відлік від 0 до фінального числа
- Кругова діаграма (SVG) з анімацією заповнення
- Зірки рейтингу — від 1 до 5 зірок з послідовною появою
- При 100% — спеціальна анімація феєрверків
- При 0% — підбадьорююча анімація (не сумна!)

### 3.5 Головне меню / Вибір теми

- Картки тем з hover-ефектом — підйом + тінь + легке свічення
- Іконки тем (emoji або SVG) з пульсацією
- Прогрес-бар під кожною темою показує пройдений рівень
- Заблоковані рівні — з ефектом "замок" та сірим затемненням
- Плавне розгортання рівнів при натисканні на тему

### 3.6 Таймер та streak

- Анімований кільцевий таймер (опціонально, для просунутих рівнів)
- Streak-лічильник (серія правильних відповідей) з вогненною анімацією
- При streak >= 3 — фон контейнера починає легко мерехтіти золотим

### 3.7 Технічна реалізація анімацій

```
CSS Animations — для простих ефектів (fade, slide, shake, pulse)
CSS Transitions — для hover, зміни кольорів, трансформацій
Canvas API    — для частинок та конфетті
SVG Animation — для кругової діаграми результатів
requestAnimationFrame — для плавних canvas-анімацій
```

## Примітки

- Проєкт залишається **статичним** (без бекенду/БД) — все працює в браузері
- Дані зберігаються в `localStorage` — при очищенні браузера дані зникнуть
- Всі анімації мають бути **плавними** і **не заважати** проходженню квізу
- Підтримка мобільних пристроїв — адаптивний дизайн
- Мова інтерфейсу — українська
